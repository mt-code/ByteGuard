=================================================================
		SELF-DISTRIBUTION (CUSTOM) README
=================================================================

This folder contains your protected application and any files that
were located within your specified application directory.

In this folder you should see a file named "ByteGuard_Protected.exe",
this is your original application after obfuscation. Any attempts
to open this file before a successful log-in will fail.

In order to use your application with a custom login interface, you
will be required to reference the library "ByteGuard.Helper.dll"
located within the folder named "Helper_Files". For more detailed
examples using this library, please visit the following link:

http://www.byteguardsoftware.co.uk/replaceme
